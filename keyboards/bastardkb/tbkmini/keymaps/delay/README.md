# My keyboard layout for the [TBK Mini](https://bastardkb.com/keebs/)

## Base: DVORAK with [home row mods](https://precondition.github.io/home-row-mods)

- `OSM_CAPS`: one shot mod caps lock

```
,-----------------------------------------------------.        ,-----------------------------------------------------.
|  L(5)  |  ' "   |  , <   |  . >   |   p    |   y    |        |   f    |   g    |   c    |   r    |   l    |        |
|--------+--------+--------+--------+--------+--------|        |--------+--------+--------+--------+--------+--------|
|        | Alt(a) | Gui(o) | Ctl(e) |Shift(u)|   i    |        |   d    |Shift(h)| Ctl(t) | Gui(n) |   s    |        |
|--------+--------+--------+--------+--------+--------|        |--------+--------+--------+--------+--------+--------|
|        |  ; :   |   q    |   j    |   k    |   x    |        |   b    |   m    |   w    |   v    |   z    |        |
'--------------------------+--------+--------+--------'        '--------+--------+--------+--------------------------'
                           |L(3,Bsp)|                                            |L(2,Spc)|
                           '--------+--------+                          +--------+--------'
                                    |L(1,Tab)|                          | Enter  |
                                    '--------+--------+        '--------+--------'
                                             |L(4,Esc)|        |OSM_CAPS|
                                             +--------+        '--------'
```

## 3: Navigation & System

- `LANG_PRV`: last used language layout
- `LANG_NXT`: next language layout in the list
- `SCRN_PRV`: previous workspace
- `SCRN_NXT`: next workspace

```
,-----------------------------------------------------.        ,-----------------------------------------------------.
|        |        |  HOME  |  PGDN  |  PGUP  |  END   |        |        |        |        |        |        |        |
|--------+--------+--------+--------+--------+--------|        |--------+--------+--------+--------+--------+--------|
|        |        |  LEFT  |  DOWN  |   UP   |  RGHT  |        |  CUT   |  COPY  |  PSTE  |        |        |        |
|--------+--------+--------+--------+--------+--------|        |--------+--------+--------+--------+--------+--------|
|        |        |LANG_PRV|SCRN_PRV|SCRN_NXT|LANG_NXT|        |        |  UNDO  |  REDO  |        |        |        |
```
